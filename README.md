# NUSH Marketing Agency

## Formularz kontaktowy

Formularz wysyła wiadomości przez SMTP z produkcyjnego serwera Node. W Railway ustaw zmienne z pliku `.env.example`:

- `SMTP_HOST`
- `SMTP_PORT`
- `SMTP_SECURE`
- `SMTP_USER`
- `SMTP_PASS`
- `CONTACT_FROM`
- `CONTACT_TO`

Nie commituj pliku `.env` ani hasła SMTP.
